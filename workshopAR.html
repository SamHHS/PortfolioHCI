Unreal Engine 4

Tijdens de workshop voor AR&VR zijn we bezig geweest met het realiseren van een banaan aan de hand van unreal engine 4. Hiervoor moesten we eerste het programma downloaden wat een tijd duurde dus het was handig als je dit al had gedaan voordat je bij de les aanwezig was.


Tijdens de les kregen we een handleiding die we moesten volgen, aan de hand van deze handleiding kreeg je meer informatie over unreal engine en werden de stappen uitgelegd die je moet volgen. Het maken van een banaan via unreal engine was leuk om te doen ondanks dat je alle textures en shapes al kreeg.


Om te beginnen moest je een nieuw project opzetten, dit werd gedaan via de software unreal engine. Hier kan je het project een naam geven zodat het duidelijk is waarvoor het gaat dienen. (scr. 2) Vervolgens pak je de verkregen bestanden uit en kan je die importeren binnen unreal engine, (scr. 3) wanneer je de bestanden geïmporteerd heb, heb je eigenlijk nog niks behalve de optie om e 2 bestanden te gebruiken binnen je project. Om het te gebruiken moet je wel een material maken van de textures die je hebt verkregen. Dit is dus de banana Texture file die je hebt gekregen. Dus wanneer je een nieuwe material wilt maken, moet je deze eerst toevoegen waarna je texture file eraan kan koppelen zodat je het kan gebruiken (scr 4).

Wanneer de textuur toegevoegd is kan je het model van de banaan binnen de workplace slepen, hierdoor plaats je het model eigenlijk op de ondergrond. Wanneer het model toegevoegd is komt het er wit/grijs en zonder texture in het programma. DIt is waar je de material toe kan voegen om het een banaan te maken. In de opties rechts in het scherm kan je dus de material selecteren waarna dit ook toegevoegd wordt en je daadwerkelijk een banaan hebt. (scr 5).


Vervolgens ging het over het toevoegen van een animatie waarbij de banaan een rondje kan draaien. Hiervoor moet je aan het model een actie koppelen. DIt doe je door het te selecteren en een blueprint ervoor te maken. Bij de blueprint kan je er een actie aan koppelen zoals het “rotaten” van de banaan. (scr 6.) Wanneer dit gedaan is heb je een banaan die rond kan draaien als een blueprint. Deze banaan kan je ook weer in het veld slepen en vervolgens met de “play” knop in het menu laten draaien (scr 7.)


Bij het a-frame moesten we werken met shapes, zoals in opdracht 02. Dit is iets om virtueel dingen te zien zodat je met de telefoon bijvoorbeeld rond kan draaien en dat er achter je iets heel anders staat dan voor je. Bij stap 6 van opdracht 02 moesten we eigen shapes toevoegen aan de virtuele ruimte. Dit doe je door de class toe te voegen en daarna de coördinaten waar je het wilt hebben. Je kan ook nog de grote van het object veranderen. Deze objecten kan je ook animeren, zodat het beweegt wanneer je ermee sleept of wanneer je erop klikt dat het sneller gaat draaien. Dit is iets dat gedaan is tijdens opdracht 04 stap 7. Zo kan je bijvoorbeeld het object een soort kickflip laten doen waarbij het om meerdere asses draait. ( scr 10 en 11)
